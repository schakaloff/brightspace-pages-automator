Using Page Automator

A plain-language walkthrough for moving Moodle course content into Brightspace, step by step. No technical background needed.

Step by step

Work through these in order the first time. Once you're comfortable, you can jump straight to the tool you need.

1 · Before you start

A few quick checks that save time later.

  1. Make sure you can log into Brightspace
    • Open Brightspace in your browser and sign in normally.
    • If you can reach your course, you're set.
  2. Make sure you can log into Moodle
    • Open Moodle and sign in.
    • Confirm you can open the course you're moving.
  3. If you'll use Kaltura, have those details saved
    • Only needed if the course has Kaltura videos.
    • You'll add them in Settings (next section).
  4. Check the Moodle course format
    • If the Moodle course uses Tab View, change the course format first.
    • In Moodle: Settings → Course format → choose Custom sections.
    Do this before scanning. Tab View can hide sections from the tools, so content may look missing when it isn't.

2 · Settings

Enter your sign-in details once. The app remembers them between sessions.

Keep it private. Never share your passwords or API key, and don't paste them into emails or chat. If you're unsure what to enter, ask your project lead before changing anything.

  1. Open Settings
    • Click Settings at the bottom of the left sidebar.
  2. Enter your Brightspace credentials
    • Type your Brightspace username and password.
    • Microsoft (MS / SSO) sign-in is not needed.
  3. Enter your Moodle credentials
    • Used by the Checker and Kaltura tools to read the Moodle course.
  4. Enter Kaltura credentials — only if using Kaltura
    • Skip this if the course has no Kaltura videos.
  5. Enter your Claude API key — only if required
    • Needed for the styling step in Unit Collector.
    • If you don't have one, ask your project lead.
  6. Choose the Claude model
    • Pick the model marked as recommended in the list.
    • Your team may specify a particular one (for example, Haiku 4.5) — use that if told to.
  7. Save your settings
    • Click Save. You only need to do this again if a password changes.

3 · Content Checker

Find out what made it into Brightspace and what didn't.

The Checker compares your Moodle course with your Brightspace course and reports what's missing. It can download missing files and H5P activities and place them into the matching Brightspace section, in the course's main content files. It also flags links and fuzzy matches — items that look like the same thing but aren't an exact name match.

  1. Paste the Moodle course URL
    • Copy the address of the Moodle course from your browser.
  2. Paste the Brightspace course or unit URL
    • Copy the address of the matching Brightspace course.
  3. Run the check
    • Start the scan and let it finish.
  4. Review the results
    • Exact — already in Brightspace, nothing to do.
    • Missing — not found in Brightspace.
    • Review / fuzzy — a possible match to double-check.
  5. Choose what to upload
    • If the missing / review popup appears, tick only the files you actually want uploaded.
    For a fuzzy match, only tick it if it's genuinely the same file. When in doubt, leave it unticked and check by hand.
  6. Let the app finish
    • Wait until uploading is complete before moving on.
  7. Read the summary
    • Confirm what was uploaded and note anything flagged for manual follow-up.

4 · Unit Collector

Combine the pages of one Brightspace unit into a single page.

The Collector gathers the content pages in one unit and merges them into a single combined page. It does not move quizzes or assignments. It can create a fresh target page for you, or fill a blank page you point it to.

  1. Paste the Brightspace unit URL
    • Copy the address of the unit you want to combine.
  2. Set the target page
    • Leave auto-create target page on to have a new blank page made for you.
    • Or make a blank page yourself first and paste its link into Target Page — handy if you want to compare before deleting anything.
  3. Choose a theme colour
    • Pick the colour theme for the page. Your department may have a preferred one (for example, EA uses Lake).
  4. Choose the speed
    • Set parallel pages to 2 or 3. Higher isn't always better and can trip Brightspace limits.
  5. Click Collect & Assemble
    • Start the run.
  6. Log in if prompted
    • If a browser window asks you to sign in, do it, then let the run continue.
  7. Wait until it's complete
    • Watch the log at the bottom. Let it finish before touching the course.
  8. Review the combined page
    • Open the new page and check everything came across.

Always review before deleting. Check the combined page carefully before you remove any original unit content. Deleting first means there's nothing to fall back on.

Very large units: if a unit is too big, the app will skip the automatic styling step on purpose. That's a safety measure to avoid wiping content — the combined text is still collected; it just won't be restyled.

5 · Kaltura

Move Kaltura videos from Moodle into Brightspace pages.

The Kaltura tool finds Kaltura videos in the Moodle course (including videos inside Books), suggests which Brightspace unit each one belongs in, and creates Brightspace pages with the videos embedded. Videos are linked in place — nothing is downloaded.

  1. Paste the Moodle course URL
  2. Paste the Brightspace course URL
  3. Find the videos
    • Click Find Videos & Suggest Destinations if available — it scans and suggests mappings in one step.
    • The first time, you may be asked to log into Moodle in the browser window.
  4. Review the suggested mappings
    • Check where each Moodle section will send its videos in Brightspace.
    • Fix any mapping marked Needs review.
    If a video's destination is unclear, leave it unticked and ask for a second opinion rather than guessing.
  5. Select the videos to transfer
    • Tick the videos you want. Use Select All if you want them all.
  6. Click Create Pages
    • Confirm when asked — this writes new pages into the live Brightspace course.
    • Log into Kaltura if prompted.
    Don't create pages until the mappings look right. This step writes directly to Brightspace and can't be undone automatically.

6 · H5P

Move H5P activities from Moodle into Brightspace.

The H5P tool finds H5P activities in the Moodle course and helps you bring them into Brightspace. Use it when the Content Checker reports H5P items that are missing.

  1. Paste the Moodle course URL
  2. Paste the Brightspace course URL
  3. Start the scan
    • Click Scrape and follow the prompts.
    • You may be asked to switch your Moodle role to Teacher so the files can be downloaded.

7 · Quality control checklist

Run through this before you delete any original content.

  • Is this the correct Brightspace unit?
  • Does the page title match the Moodle name?
  • Do the files open?
  • Do the videos play?
  • Does the H5P activity work?
  • Do the links open?
  • Does the combined page have all the expected content?
  • Are there no duplicate pages?
  • Old content deleted only after this review?

8 · Troubleshooting

Common situations and what to do.

  • The app asks me to log in again

    Sign in when the browser window appears, then start the run again.

  • A missing-file popup appeared

    Review the ticked files before uploading. Only tick what you actually want added.

  • A fuzzy match showed up

    Only select it if it really is the same file. If you're not sure, leave it and check by hand.

  • The Kaltura scan is taking a long time

    That's normal for courses with Books that have many chapters — it can take several minutes.

  • The page looks empty after collecting

    Stop and report it. Do not delete the original content — the original is your backup.

  • It said upload failed, but the file appears later

    Refresh Brightspace and check the module. The upload may have finished after the message.

Manual fallback

Step-by-step instructions for doing by hand what the app usually does automatically. Use these if a run fails partway through.

Move one Kaltura video by hand

Manual steps for moving one Kaltura video from a Moodle course into the matching Brightspace unit.

  1. Find the video in Moodle
    • Open the Moodle course.
    • Find the video item.
    • Write down:
      • video title
      • Moodle section name
  2. Find the matching Brightspace unit
    • Open the Brightspace course.
    • Find the unit with the same or similar section name.
    • If unsure, stop.
  3. Create a page
    • Click Create New
    • Choose Page
    • Title it exactly like the Moodle video.
  4. Insert the Kaltura video — preferred way
    • Click Insert Stuff
    • Choose Kaltura / My Media / Kaltura Media
    • Search the exact video title.
    • Select the video.
    • Insert it.
  5. If the Kaltura option isn't available
    • Use KMC.
    • Search the video.
    • Use Share & Embed.
    • Prefer an iframe embed, not a JavaScript/script embed.
    • Paste only inside Source Code — never the normal editor.
    Pasting into the normal editor box shows the code as plain text instead of a video. Always use Source Code.
  6. Save and check
    • Click Save and Close.
    • Open the page.
    • Confirm the video appears and plays.
