Phase 1 — Checker Tab

Compares your Moodle course against Brightspace and fills in everything that's missing.

1

Log in to Brightspace

The app opens a browser window. Sign in with your Okanagan College account and complete MFA. The script waits automatically until you reach the home page.

⏸

Pause — Navigate to your Moodle course

A second browser tab opens Moodle. Navigate to the correct course page, then click ✅ Ready — Scrape Now in the app. This is where you confirm the right course before anything is read.

2

Scrape Moodle course structure

The app reads every section, file, link, quiz, and H5P activity from your Moodle course and builds a complete inventory.

⏸

Pause — Verify instructor role for H5P

H5P downloads require Teacher access in Moodle. The app tries to switch roles automatically — if it can't, it stops here so you can confirm the browser looks right before clicking ✅ Ready — Download H5P.

3

Download H5P activities

Each H5P activity (drag-the-words, flashcards, quizzes, etc.) is downloaded as a .h5p file. Already-downloaded files are skipped automatically.

4

Compare Moodle vs Brightspace

Every Moodle item is matched against the Brightspace course — exact matches, fuzzy matches, and missing items are all identified. A full report is shown in the log.

⏸

Pause — Select files to download

A checklist appears showing all files that are missing from Brightspace. You choose which ones to include — some may be intentionally absent. Click Download Selected or Skip All.

5

Download & upload missing files

Selected files are downloaded from Moodle, renamed to match their original course names, then uploaded to the correct Brightspace modules. Each upload is verified via the D2L API before a ✓ is shown.

6

H5P Phase A — Upload to cloud

All H5P files for this course are uploaded to the H5P cloud (h5p.com). Files already in the cloud are detected in one scan and skipped — only missing ones are uploaded.

⏸

Pause — Confirm H5P insertion

Phase A is complete and everything is in the cloud. Before Phase B embeds the activities into Brightspace pages, the app pauses so you can verify the cloud uploads look correct.

7

H5P Phase B — Insert into Brightspace

A new Brightspace page is created in the correct module for each H5P activity, and the activity is embedded from the cloud. Activities already present are detected live and skipped.

8

Migration summary

A full summary is shown: time per phase, files transferred, H5P embeds succeeded/failed. You're prompted whether to clear the downloads folder to free up space.

✅ Verify all content in Brightspace before continuing →
📦

Phase 2 — Unit Collector Tab

Combines all topics in a unit into one clean, collapsible summary page.

⏸

Before starting — confirm content is correct

Unit Collector overwrites the target page. Make sure all files, H5P activities, and links are correct in Brightspace first. This is the last chance to catch anything before the unit is assembled.

1

Enter the Brightspace unit URL

Paste the URL of the unit (module) you want to collect. The app navigates to each topic in that unit and scrapes its content.

2

Collect & combine all topics

Every topic page in the unit is read and merged into a single HTML document with collapsible sections — one section per topic.

3

Write to target page

The combined page is written to a designated blank Brightspace page, giving students one place to read the entire unit.

Phase 3 — Page Changer Tab

Uses AI (Gemini) to restyle Brightspace pages to match the OC brand — in bulk.

1

Paste a Brightspace section or page URL

For a single page, paste the topic URL. For batch processing, paste the section URL — the app finds and processes all topics in that section automatically.

2

Choose an OC brand theme

Pick from 9 official Okanagan College brand colours: Lake (teal), Sky, Sunset, Peach, Cherry, Cabernet, Lavender, Lilac, or Charcoal. Lake is the recommended default.

3

AI restyling runs automatically

Gemini AI reads the existing page HTML and rewrites the layout to match the selected OC theme — preserving all content while adding proper spacing, typography, and branded card styles.

4

Styled HTML is written back

The new HTML is injected into the Brightspace page editor and saved. The original content is never deleted — only the presentation is updated.

📌 Things to keep in mind

  • Always run in order — Checker first, then Unit Collector, then Page Changer.
  • The Checker can be re-run safely — it skips anything already in Brightspace or already downloaded.
  • H5P content types must be installed in Brightspace Admin → H5P → Content Type Hub before Phase A will work.
  • Clear the downloads/ folder when switching to a new course — old files from previous courses can interfere.
  • Page Changer can be run on individual pages or entire sections — start with one page to verify the style before doing a batch.
  • Session cookies are saved after login — you won't be asked to log in again until the session expires or you delete session.json.
