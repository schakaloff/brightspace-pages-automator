Phase 1 — Checker

Finds everything missing from Brightspace and migrates it automatically.

Log in to Brightspace

The app opens a browser window. Sign in with your OC account and complete MFA. The script continues automatically once you reach the home page.

⏸ Pause — Navigate to your Moodle course

A Moodle tab opens. Go to the correct course page, then click ✅ Ready — Scrape Now in the app. This confirms you're on the right course before anything is read.

Scrape Moodle course structure

Every section, file, link, quiz, and H5P activity is read from Moodle and catalogued.

⏸ Pause — Verify instructor role for H5P

H5P downloads require Teacher access. The app tries to switch roles automatically — if it can't, it stops here for you to confirm before clicking ✅ Ready — Download H5P.

Download H5P activities

Each H5P activity (drag-the-words, flashcards, quizzes, etc.) is saved as a .h5p file. Files already downloaded are skipped.

Compare Moodle vs Brightspace

Every Moodle item is matched against Brightspace — exact matches, fuzzy matches, and missing items are all identified. A full report is shown.

⏸ Pause — Select files to download

A checklist shows all files missing from Brightspace. You choose which to include — some may be intentionally absent. Click Download Selected or Skip All.

Download & upload missing files

Selected files are downloaded from Moodle, renamed to their original course names, and uploaded to the correct Brightspace modules. Each upload is verified via the D2L API.

H5P Phase A — Upload to cloud

All H5P files for this course are scanned against the H5P cloud in one pass. Only files that aren't there yet are uploaded.

⏸ Pause — Confirm H5P insertion

Everything is in the cloud. Before Phase B embeds the activities into Brightspace pages, the app pauses so you can verify the uploads look correct.

H5P Phase B — Insert into Brightspace

A Brightspace page is created in the correct module for each H5P activity. Activities already present are detected live and skipped automatically.

Migration summary + cleanup prompt

A full summary shows time per phase, files transferred, and H5P embeds. You're asked whether to clear the downloads folder to free up space.

📦

Phase 2 — Unit Collector

Combines all topics in a Brightspace unit into one clean, collapsible summary page.

⏸ Before starting — confirm all content is correct

Unit Collector overwrites the target page. Make sure all files, H5P activities, and links from Phase 1 are correct in Brightspace first. This is the last chance to catch anything.

Enter the Brightspace unit URL

Paste the URL of the unit (module) you want to collect. The app navigates to each topic inside it.

Collect & combine all topics

Every topic page in the unit is read and merged into a single HTML document with collapsible sections — one section per topic.

Write to target page

The combined page is written to a designated blank Brightspace page, giving students one place to access the entire unit.

⚡

Phase 3 — Page Changer

Uses Gemini AI to restyle Brightspace pages to match the official OC brand — one page or a full section at once.

Paste a Brightspace section or page URL

For one page, paste the topic URL. For batch mode, paste the section URL — all topics in that section are styled automatically.

Choose an OC brand theme

Pick from 9 official Okanagan College colours: Lake, Sky, Sunset, Peach, Cherry, Cabernet, Lavender, Lilac, or Charcoal. Lake is the recommended default.

Gemini AI restyling runs

AI reads the existing page HTML and rewrites the layout to match the chosen OC theme — preserving all content while adding proper spacing, typography, and branded card styles.

Styled HTML is saved back to Brightspace

The new HTML is injected into the Brightspace editor and saved. The original content is never deleted — only the visual presentation is updated.

📌 Good to know

  • Run in order — Checker first, then Unit Collector, then Page Changer.
  • The Checker is safe to re-run — it skips anything already in Brightspace or already downloaded.
  • H5P content types must be installed in Brightspace Admin → H5P → Content Type Hub before Phase A will work.
  • Clear the downloads folder between courses — leftover files from a previous course can interfere.
  • Session cookies are saved after login — you won't be asked to log in again until the session expires.
  • Try Page Changer on one page first to verify the style before running it on a full section.
